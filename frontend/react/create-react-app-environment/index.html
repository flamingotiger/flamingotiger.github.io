create-react-app에서 env/환경변수 사용하기

2019-12-16
create-react-app, frontend, react
Views: 274

시작하기

환경변수를 사용해야하는 경우가 있습니다.
Redux devtool을 예로 들수있습니다. 개발자의 경우에는 디버깅시에 필요하지만 사용자는 보면 안되는 데이터가 있을수 있습니다.
production 과 development에 따라 구분해서 보여줄 필요가 있습니다.

  • create react app은 환경변수를 사용하기 위해서 dotenv(https://www.npmjs.com/package/dotenv) 모듈을 사용하고 있습니다.

env 파일 구성

env파일은 프로젝트 루트에 존재해야합니다.

env 실행 우선순위

왼쪽이 오른쪽보다 우선순위가 높습니다.

사용예시

Create react app에서는 변수명앞에 무조건 REACT_APP_가 있어야 인식을 합니다.
사용하지 않을시 변수를 무시합니다.

현재 상태를 구분하기 쉬운 NODE_ENV

이런 방식으로 사용할수 있습니다.

자세한 내용은 create-react-app docs: Adding Custom Environment Variables에서 확인 할 수 있습니다.
