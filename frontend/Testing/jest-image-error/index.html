Jest 이미지 테스트 에러

2020-02-11
config, jest
Views: 108

시작하기

jest 테스트를 하던 도중 이미지 import image from './image.svg'; 부분이 에러가 발생시 해결방법입니다.

설정파일 만들기

프로젝트의 루트폴더에 assetsTransformer.js 파일을 생성해줍니다.

jest 설정 수정하기

jest.config.js 혹은 package.json 파일에 추가 해줍니다.
이후 테스트를 하면 에러가 없어지는 것을 확인 할 수 있습니다.
